Qué hace cada parte

Suscribir arriba llama a la API de Alegra POST /webhooks/subscriptions y solo registra la URL de tu túnel; no crea facturas ni envía datos de negocio.

Cuando ocurre un evento, Alegra hace POST a la URL registrada (host + {{ webhook_ingest_suffix }} + ?empresa=<NIT> al suscribir desde aquí). El cuerpo se guarda en Recepciones y, si el JSON es válido, se crea o actualiza accounting.Facturas (ver alegra_integration/webhook_bills.py).
